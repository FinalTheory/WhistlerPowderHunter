WHISTLER SENSOR STATION DATA

Purpose of sensor station data:

- These station observations are provided to validate short-term model forecasts.
- Use them to assess whether real-world temperature and wind trends are:
  - Consistent with model expectations, or
  - Diverging from the forecast signal.

Interpretation rules:
- Compare observed trends (temperature, avg/max wind, direction shifts) against the expected evolution implied by the forecast models.
- If a discrepancy exists, explicitly state:
  - Whether the observed change is occurring EARLIER than forecast, LATER than forecast,
  - Or evolving in a DIFFERENT direction (e.g., warming instead of cooling, wind strengthening vs weakening).
- Focus on trend timing and direction, not exact numeric matching.
- Any detected divergence should be clearly highlighted in the output, as it may affect short-term confidence and monitoring frequency.

Sensor stations data format:

- Input is a dictionary: { "<station_name>": [ "<reading_string>", ... ], ... }.
- Each station maps to a list of human-readable readings.
- IMPORTANT: For every station, the list is in reverse chronological order (most recent first).
- Each reading string contains:
  - Local (PST) time label (e.g., "10:15PM")
  - Temperature in °C (e.g., "Temperature=-5.6C")
  - Wind direction as both compass + degrees (e.g., "Wind Direction=NNE (13degree)")
  - Wind speeds in km/h: Max Wind and Avg Wind (e.g., "Max Wind=28.1km/h Avg Wind=19.7km/h")

Parsing rules:
- Station names imply relative terrain exposure and elevation:
  - "peak", "harmony", "symphony", "horstman"(7th heaven top): exposed alpine
  - "roundhouse", "rendezvous": upper to mid-mountain
  - "crystal": lower or more sheltered terrain
- Use the reverse-time ordering when computing trends (e.g., increasing wind means values grow when moving forward in time, i.e., from the end of the list toward the start).
- Do not assume missing metadata (elevation/coordinates) unless explicitly provided.

   - "WARM_STORM": warm system, high snow line, rain, or snow-quality degradation likely.
